# Security Policy

## Supported Versions

Only the latest tagged release is actively supported for security fixes.

## Reporting a Vulnerability

Please report security issues privately instead of opening a public issue.

- Contact: `security@ulbo.dev` (placeholder)
- Include: affected version, reproduction steps, impact assessment

You can expect an initial response within 3 business days.

## Disclosure Process

1. We confirm and triage the report.
2. We prepare and test a fix.
3. We publish a patched release and disclosure notes.
