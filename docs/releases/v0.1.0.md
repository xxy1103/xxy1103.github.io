# v0.1.0 Release Notes

Release date: 2026-02-10

## Highlights

- Centralized all user-facing configuration under `src/config/`.
- Added feature flags for default custom cursor and build-time image optimization.
- Removed bundled personal blog content and legacy `public/` assets.
- Ensured zero-content safety for `/`, `/blog`, `/tags`, `/about`, RSS, and search index.
- Added open-source governance files and issue templates.
- Added CI workflow and updated Pages deploy workflow for `main`.

## Migration Notes

1. Update values in:
- `src/config/site.ts`
- `src/config/profile.ts`
- `src/config/hero.ts`
- `src/config/features.mjs`

2. Add your posts to `src/content/blog/`.
3. If needed, disable default image optimization in `src/config/features.mjs`.

## Verification Checklist

- `npm ci`
- `npm run build`
- `npm run preview`

## Known Manual Follow-ups

- Create and configure the GitHub repository as a template repository.
- Submit to Astro Themes through the official portal.
