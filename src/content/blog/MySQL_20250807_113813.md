### cmd

使用win自带的终端如cmd，powershell连接mysql需要配置环境变量：

在Path中添加这一路径：

![1754542668095](image/MySQL_20250807_113813/1754542668095.webp)

连接命令：

```bash
mysql [-h 127.0.0.1] [-P 3306] -u root -p
```

* -h：连接的主机ip
* -P：连接的端口号
* -u：登录用户
* -p：输入密码验证

> `[]`中的为默认配置，不修改时可以省略

# MySQL数据库

## 关系型数据库(RDBMS)

**概念**：建立在关系模型基础上，由多张相互连接的**二维表**组成的数据库。

**特点**：

* 使用**表存储**数据，格式统一。
* 使用**SQL语言**操作，标准统一。

**数据模型**：

![1754551365889](image/MySQL_20250807_113813/1754551365889.webp)

* **DBMS(数据库管理系统)**：
  * DBMS是位于客户端和数据库之间的软件层。主要用于管理和控制数据库的访问。
  * DBMS负责处理数据的存储、检索、安全、完整性和并发控制等任务。
* **数据库**：
  * 数据库是数据的物理存储位置，它由许多表组成。
* **数据表**：
  * 这是关系型数据库的核心。数据都以表格的形式组织，由行(记录)和列(字段)组成。
* **客户端**：客户端负责用户交互，DBMS负责数据管理和处理，数据库负责数据的实际存储，职责分离。
