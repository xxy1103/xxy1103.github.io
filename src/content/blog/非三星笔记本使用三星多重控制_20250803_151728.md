# Multi Control

![1754205754718](image/非三星笔记本使用三星多重控制_20250803_151728/1754205754718.webp)

给平板刷上了三星最新推出的OneUI7系统在探索新功能时，却发现了一个以前从来没有体验过的老功能：Multi Control。这个功能可以让你的**电脑**，**平板**，**手机**共用一套键鼠。但是这个功能却不支持非三星GalaxyBook之外的其他Win11电脑。

这里记录一下在我的电脑上，我是如何实现使用三星多重控制的。博主测试了B站，xda等处的多篇教程。着重强调几处其他教程中没有明确指出的细节。

## galaxybook_mask

Github地址：[kellwinr/galaxybook_mask](https://github.com/kellwinr/galaxybook_mask)

galaxybook_mask是一个bat脚本，下载后**执行一次**即可。执行一次后脚本会自动添加到开机自启动，在每次开机时自动执行一次，修改电脑注册表，对于软件伪装成为一个GalaxyBook。

# Quick Share

使用海外版安卓系统的朋友应该熟悉Quick Share这个功能，QuickShare是Google推出的无线文件传输工具。在较新的android系统中内置该功能，可以方便的给周围的设备互传文件。

![1754209134382](image/非三星笔记本使用三星多重控制_20250803_151728/1754209134382.webp)

如何优雅的给电脑互传文件呢？

通过QQ或者微信转发虽然并不麻烦但是一点也不优雅，连接到Windows只能给已连接的Windows发送文件。

QuickShare完美的解决这个问题，给电脑传文件就像手机间互传一样优雅。

# OneDrive

oneDrive是微软推出的云存储服务，允许你将文件、照片、视频等内容上传到云端，并在任何设备上访问、编辑和共享这些文件。

**存储空间**：

| 账户类型             | 云存储空间        |
| -------------------- | ----------------- |
| 免费 Microsoft 账户  | 5 GB              |
| Microsoft 365 基础版 | 100 GB            |
| Microsoft 365 个人版 | 1 TB（单人）      |
| Microsoft 365 家庭版 | 最多 6 TB（多人） |

通常OneDrive会同步：`C:\Users\<用户名>\OneDrive`路径下的文件。

## Windows文件系统链接方式

1. **快捷方式**(.lnk)
   * **图标形式**存在，Windows用户最熟悉的一种链接方式
   * 它本质上是一个指向原始文件或位置的**小文件**。当你双击快捷方式时，系统会打开它所指向的原始目标。**但是系统读取时仅为一个 `.lnk`文件**
   * 快捷方式的功能相对有限。如果原始文件被移动、重命名或删除，快捷方式就会失效。它并不是原始文件本身，删除快捷方式不会影响原始文件。主要用于用户界面的便利性。
2. **软链接**(Symbolic Link)
   * 软链接，也称符号链接，是指向**文件路径的链接**。它允许跨分区链接，非常适合开发和资源管理。在Windows中，可以用 `mklink`命令来创建软链接。
   * 软链接的功能类似于快捷方式，但功能更强大。它是一个特殊的小文件，其中包含了它所指向的文件或者目录的路径。**当操作系统访问软链接时，它会读取路径并重新定位到目标文件目录。**
   * 软连接允许跨分区。如果原始文件或目录被删除，软连接会失效，因为它指向的路径不再存在。删除软连接本身不会影响原始文件。
3. **硬链接**(Hard Link)
   * 硬链接是指向**文件数据**本身的链接，不可跨分区。删除原始文件不会影响硬链接。
   * 与软连接不同，硬链接不是一个独立的文件，它与原始文件共享相同的数据内容和inode。可以理解为一个文件有了多个名字。
   * 硬链接由于指向文件数据，而不是文件路径，因此只要有一个硬链接存在，文件数据就不会被删除。只有当所指向的文件数据的硬链接都被删除时，文件数据才会被释放。

## 让Onedrive备份文件夹

我们可以通过创建软链接的方式，在让OneDrive备份我们电脑中的任意文件或文件夹。实现**跨设备同步**和**脱机访问**的效果。

1. 找到OneDrive的本地路径：博主的路径是：`C:\Users\ulna\OneDrive`
2. 找到想要同步**文件夹**的路径：`C:\Users\ulna\Desktop\xxyefwffre\blog`
3. 使用**管理员权限**启动cmd：输入命令

   ```bash
   mklink /D C:\Users\ulna\OneDrive\blog C:\Users\ulna\Desktop\xxyefwffre\blog
   ```

> 注意在OneDrive后添加软链接的名称
