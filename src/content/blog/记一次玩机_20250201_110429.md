# 刷机准备

**重点**：刷机之前，请一定退出手机中的所有账号，尤其是**三星**账号、**google**账号等。否则刷机后，可能导致被google锁，需要能够直连google的wifi环境才能够解锁。(血泪的教训，本人给平板刷机被锁了一次，连夜给路由器刷了openwrt，给路由器挂梯子后平板连接登录google账号才好)

重点：玩机前请一定做好数据备份，很多操作都会清除手机数据，并且遇到意外也可能需要重新刷机。

![1738379669063](image/记一次玩机_20250201_110429/1738379669063.webp)

在系统中记录的账号在刷机前最好全部退出。

# 解BL锁

* **备注**：如果你只想给手机换区，或者升级，不需要root或者刷入第三方固件，可以不用解锁BL，三星设备的BL锁不可逆，knox会物理熔断。
* **备注**：解锁一定会清除数据

所有的手机刷机前都需要解BL锁，不同品牌解锁方式不同，三星手机解锁几乎是最简单的一个。

1. 系统进入开发者模式
2. 打开OEM锁
3. 手机数据线连接电脑
4. 关机按 `电源键+音量上+音量下`，进入引导界面
5. 按照提示解锁手机（一般是长按音量上键）

# 换区\升级

三星官方固件下载(需要梯子)：[Samsung Firmware Download - Lastest official firmware update](https://samfw.com/)

Odin3下载：[Archives - Odin Downloader Tool](https://odindownloader.com/category/download)

首先我们在SamFw中搜素自己的的设备型号，如国行s24的型号为：SM-S9210

![1738381047201](image/记一次玩机_20250201_110429/1738381047201.webp)

在下方选择自己喜欢的固件进行下载：

![1738381090920](image/记一次玩机_20250201_110429/1738381090920.webp)

下载完成后将压缩包解压，一般会得到这些文件：

![1738381203826](image/记一次玩机_20250201_110429/1738381203826.webp)

如过你的设备是不带通信功能的平板，那么会少一个 `CP`文件，CP是基带文件。

我们将AP、BL、CP、CSC四个文件拖入Odin3：

如果只想系统升级，保留数据，则将CSC替换为HOME_CSC

![1738383534069](image/记一次玩机_20250201_110429/1738383534069.webp)

手机数据线连接电脑，`电源` + `音量上` + `音量下` 进入引导界面，这次进入download模式（短按一下音量下键）。

点击start进行刷入。

如果你只想换区/升级，到这里等待刷完就可以了。

# root

本人只会比较传统的Magisk修补的方法进行root。虽然是非常传统的方法了，但是我认为并不过时，并且生态丰富。

## Magisk修补镜像

* 备注：只想root也请先进行上一步升级操作，确保刷入镜像可以正常开机，保证boot镜像与系统匹配。

magisk:[topjohnwu/Magisk: The Magic Mask for Android](https://github.com/topjohnwu/Magisk)

我们将刷机包中的AP文件，传到手机上，手机上下载安装Magisk软件。

点击安装：

![1738383251325](image/记一次玩机_20250201_110429/1738383251325.webp)

点击：`选择并修补一个文件`

![1738383311144](image/记一次玩机_20250201_110429/1738383311144.webp)

然后进入文件管理器选择我们传入的AP文件，等待修补完成。

修补完成后我们会在Download目录下找到这样一个文件：

![1738383430778](image/记一次玩机_20250201_110429/1738383430778.webp)

## 刷入修补后的镜像

我们再将他传回电脑。

这次，我们用我们修补过的文件代替原本的AP文件，进行刷入：

![1738383587471](image/记一次玩机_20250201_110429/1738383587471.webp)

# root后的世界

## LSPosed

官方的LSPosed由于一些原因已经停更，表示遗憾，较高的安卓版本无法支持，可以使用第三方维护的版本：[mywalkb/LSPosed_mod: My changes to LSPosed](https://github.com/mywalkb/LSPosed_mod)

感谢还在维护的人！！

推荐使用：Zygisk模式，方便

## 去广告模块

[lingeringsound/10007: hosts of block ads](https://github.com/lingeringsound/10007)

刷入即生效，非常好用，使用之后手机广告少多了

## Shamiko

[Releases · LSPosed/LSPosed.github.io](https://github.com/LSPosed/LSPosed.github.io/releases)

![1738384587582](image/记一次玩机_20250201_110429/1738384587582.webp)

2. 在配置排除列表中选择需要隐藏的应用

## scene

![1738384727274](image/记一次玩机_20250201_110429/1738384727274.webp)

scene可以用来管理后台，配置系统调度、电池检测和充电控制、应用冻结等功能集于一体的app，功能强大，并且后台占用低：

![1738384904086](image/记一次玩机_20250201_110429/1738384904086.webp)

## Noactive

Noactive是一款第三方模拟ios墓碑实现的程序，官方网址：[首页 | NoActive](https://app.myflv.cn/)

实现的效果是将后台应用冻结，让他们不占用cpu的同时留在缓存中，节省后台点亮的同时确保下一次的启动速度。

![1738385112066](image/记一次玩机_20250201_110429/1738385112066.webp)

## KnoxPath

KnoxPath是一个LSPosed模块，可以恢复因knox熔断而无法使用的部分功能，如三星健康，安全文件夹。

![1738385216315](image/记一次玩机_20250201_110429/1738385216315.webp)
